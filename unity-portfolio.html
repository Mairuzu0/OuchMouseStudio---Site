Star Eden

Star Eden was created as an FPS game with some focus on the environemnts and the visual effects. This involved creating the environments and tweaking the lightning and shaders. Unity's shadergraph and VFX Graph were used to create the dissolve effect. I wanted to use a pixellated type effect to match the low-poly style of the game better.
